# DNSC6330-10-Individual-Week-2-Compas-analysis-python
# COMPAS Replacement Model Analysis (Week 2)

## Purpose of the Analysis
In this project, I analyzed a COMPAS replacement model to understand how it makes predictions. The main goal was to use explainability methods to see which features influence the model and how reliable those explanations are.

## Python Libraries Used
The following libraries were used in this project
- pandas  
- numpy  
- scikit-learn  
- shap  
- lime  
- dice-ml  


## Instructions to Reproduce the Results

To reproduce this analysis, follow these steps:

1. Open the Jupyter Notebook (`Compas_Analysis.ipynb`).
2. Make sure all required libraries are installed. If not, install them using:
- bash
- pip install pandas numpy scikit-learn shap lime dice-ml
